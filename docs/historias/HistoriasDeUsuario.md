# Histórias de Usuário

## 👤 Cliente

- Como cliente, quero me cadastrar no sistema para poder acessar as funcionalidades de aluguel de automóveis.

- Como cliente, quero fazer login no sistema para acessar minha conta e gerenciar meus pedidos.

- Como cliente, quero recuperar minha senha informando meu e-mail para redefinir meu acesso ao sistema caso eu esqueça minha senha.

- Como cliente, quero criar um pedido de aluguel para solicitar um automóvel.

- Como cliente, quero consultar meus pedidos para acompanhar o status do aluguel.

- Como cliente, quero modificar um pedido de aluguel para corrigir ou atualizar informações.

- Como cliente, quero cancelar um pedido de aluguel para desistir da solicitação.

---

## 🏢 Agente (Empresa/Banco)

- Como agente, quero fazer login no sistema para acessar os pedidos de aluguel.

- Como agente, quero avaliar pedidos de aluguel para analisar a viabilidade financeira do cliente.

- Como agente, quero aprovar pedidos de aluguel para permitir a execução do contrato.

- Como agente, quero reprovar pedidos de aluguel para negar solicitações inviáveis.

- Como agente, quero modificar pedidos de aluguel para ajustar informações necessárias durante a análise.

---

## ⚙️ Sistema

- Como sistema, quero garantir que apenas usuários cadastrados possam acessar as funcionalidades.

- Como sistema, quero armazenar os dados dos clientes, incluindo identificação, profissão e rendimentos.

- Como sistema, quero gerenciar os dados dos automóveis disponíveis para aluguel.

- Como sistema, quero associar pedidos de aluguel a contratos de crédito quando necessário.

- Como sistema, quero permitir que os pedidos sejam avaliados por agentes antes da aprovação final.
